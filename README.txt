Урок 1. Приложение заметки (Python)
Информация о проекте
Необходимо написать проект(консольное приложение), содержащий функционал работы с заметками.
Сохранение заметок необходимо сделать в формате json или csv формат (разделение полей рекомендуется делать через точку с запятой).

Заметка должна содержать 
-идентификатор, 
-заголовок, 
-тело заметки,
-дату/время создания или последнего изменения заметки.

Программа должна уметь:
-создавать заметку,
-сохранять заметку(сохранять данные в файл), 
-читать список заметок(выводить на экран выбранную запись, делать выборку по дате, выводить на экран весь список записок, читать данные из файла), 
-редактировать заметку, 
-удалять заметку.

Как сдавать проект
Для сдачи проекта необходимо создать отдельный общедоступный репозиторий (Github, gitlub, или Bitbucket). 
Разработку вести в этом репозитории, использовать пул реквесты на изменения. 
Программа должна запускаться и работать, ошибок при выполнении программы быть не должно.
